# Codex Review Request: アールライバルズ オンライン

## 概要
2人対戦カードゲーム「アールライバルズ」のオンライン版。Node.js + Express + Socket.io で構築。

## レビュー対象ファイル
- `server.js` — ゲームサーバー（ルーム管理、ゲームロジック、Socket.io通信）
- `public/index.html` — ゲームUI（ロビー・ゲーム・ゲームオーバー画面）
- `public/style.css` — スタイリング（ダークテーマ、カードデザイン、アニメーション）
- `public/game.js` — クライアント側ゲームロジック（Socket.io通信、UI制御）

## アーキテクチャ
- サーバー: Express + Socket.io、メモリ内ルーム管理
- クライアント: バニラJS、Socket.ioクライアント
- ゲームフロー: ルーム作成/参加 → 同時カード選択 → ラウンド解決 → 4点先取で勝利

## レビューしてほしいポイント
1. **ゲームロジックの正確性** — カード効果（姫vs王子、暗殺者vs王子、僧侶の得点減少、魔術師の覗き見）が正しく実装されているか
2. **セキュリティ** — チート対策（手札情報の秘匿、不正なカード選択の防止）
3. **Socket.io通信** — 切断処理、レースコンディション、状態の同期
4. **UX** — ゲームフローの自然さ、エラーハンドリング
5. **コード品質** — 構造、可読性、改善提案
