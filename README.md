# R-Rivals Online

2人対戦カードゲーム「アールライバルズ」のオンライン版。
ブラウザ上でCPU対戦・オンライン対人対戦ができる。

## 起動方法

```bash
node server.js
```

ブラウザで `http://localhost:3000` を開く。

> Node.js が必要。`node_modules` は同梱済みなので `npm install` 不要。

## ゲーム概要

各プレイヤーは8枚のカード（0〜7）を持ち、1枚ずつ出して勝負。先に4勝した方が勝ち。

### カード一覧

| # | 名前 | 能力 |
|---|------|------|
| 0 | 道化 | 勝敗を次の勝負に持ち越す |
| 1 | 姫 | 相手が王子なら即ゲーム勝利 |
| 2 | 密偵 | 次の勝負で相手が先出し |
| 3 | 暗殺者 | 数字の強弱を逆転（王子は無効化） |
| 4 | 大臣 | 勝利で2勝分 |
| 5 | 魔術師 | 相手の能力を無効化 |
| 6 | 将軍 | 次に出すカードの強さ+2 |
| 7 | 王子 | 最高数値（能力なし） |

### 追加ルール（7種実装済み）

| ルール | 内容 |
|--------|------|
| トレイター | 1枚捨てて残り7枚を相手と交換 |
| ルーザー | 相手の勝ち数を4にしたら勝利 |
| フォーチューン | 山札から毎回1枚めくって対決 |
| スリーカード | 手元3枚から1枚選んで対決 |
| ランダムディール | 16枚をシャッフルして8枚ずつ配る |
| インペイシェンス | 3勝先取で決着 |
| ナンバーズ | 道化と大臣の能力のみ使用 |

## 対戦モード

- **CPU対戦**: 1人でCPU相手に練習
- **オンライン対戦**: 合言葉（4文字コード）で友達と対戦

## 技術構成

- **サーバー**: Node.js + Express + Socket.io
- **フロント**: HTML/CSS/JS（フレームワークなし）
- **アセット**: カードスプライトシート（4x2）、BGM/SE（MP3）

```
│   ├── style.css      # スタイル
│   ├── game.js        # クライアントロジック
│   ├── images/        # カード・背景・ボタン画像
│   └── audio/         # BGM・効果音（MP3）

```
